# CodSoft Data Science Internship

This repository contains the machine learning tasks completed during my **Data Science Internship at CodSoft**. The projects focus on applying data analysis, preprocessing, and classification techniques using Python and popular data science libraries.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Task 1: Titanic Survival Prediction

A classification model built using the Titanic dataset to predict whether a passenger survived or not based on features such as age, gender, ticket class, and fare.

## Task 2: Iris Flower Classification

A machine learning model developed using the Iris dataset to classify flowers into three species: Setosa, Versicolor, and Virginica based on sepal and petal measurements.

## Task 3: Credit Card Fraud Detection

A fraud detection model that identifies fraudulent credit card transactions. The dataset was preprocessed, normalized, and handled for class imbalance before training classification algorithms and evaluating performance using precision, recall, and F1-score.

## Learning Outcomes

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Machine learning model development
* Model evaluation using performance metrics
